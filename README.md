# E-Commerce Platform

A full-stack e-commerce application built with Spring Boot (backend) and React (frontend).

## Tech Stack
- Backend: Java, Spring Boot, Spring Data JPA, MySQL
- Frontend: React

## Project Status
Initial setup and backend design phase.

## Responsibilities
- Backend architecture, APIs, and database design: Owner
- Frontend implementation: Collaborative

## Goals
- User authentication and authorization
- Product and category management
- Cart and order handling
